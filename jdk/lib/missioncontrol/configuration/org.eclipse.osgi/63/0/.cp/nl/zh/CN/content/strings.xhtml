JMX 控制台是一个用于监视和管理正在运行的 JVM 实例的工具。此工具显示有关内存使用量和 CPU 占用率、垃圾收集、线程活动等的活动数据。它还包括一个全功能的 JMX MBean 浏览器，使用该浏览器可监视和管理 JVM 和 Java 应用程序中的 MBean。
Java 飞行记录器生成有关 HotSpot JVM 及其运行的应用程序的详细记录。您可以使用飞行记录器工具对记录的概要文件进行脱机分析。记录的数据包括方法和锁的概要分析、垃圾收集统计信息、优化决策和事件等待时间等。
Oracle Java Mission Control 是一个用于对 Java 应用程序进行管理、监视、概要分析和故障排除的工具套件。首次安装时，Java Mission Control 包括 JMX 控制台和 Java 飞行记录器。从 Mission Control 中可以轻松安装更多插件。
存在一些扩展 Mission Control 功能的可用插件。所有这些插件均可从 Mission Control 中安装。转至“帮助”->“安装新软件”菜单选择要安装的插件！下面提供了可用插件的一些示例：
